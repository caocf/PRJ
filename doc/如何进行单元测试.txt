1. 目前只有service及dao层支持单元测试
2. 步骤：
   a. 在需要进行测试的Spring Bean中添加不带参数，返回void的测试方法
   b. 在测试方法添加相应的标签
            如果是service层代码，添加@Test标签
            如果是dao层代码，添加@Test和@Transactional标签
    
		例如
		com.example.service.impl.ExampleServiceImpl.java
		/**
		 * Junit4  单元测试,不带参数
		 */
		@Test
		public void getExamples() {
			System.out.println("size"+this.getExamples(-1, -1).getData().size());
		}
		
		com.example.dao.impl.ExampleDaoImpl.java
		/**
		 * Junit4  单元测试,不带参数
		 */
		@Test
		@Transactional
		public void getExamples() {
			System.out.println("size"+this.getExamples(-1, -1).getData().size());
		}
	c. 在该方法上右键 run as JUnit Test
